Interfaz Mitutoyo Bluetooth

Diseñado para aplicaciones de medición y certificaciones la Interfaz Bluetutoyo brinda una solución que agiliza el proceso de registro de datos.

La interfaz se integra los con indicadores Mitutoyo, manteniendo todos los beneficios de estos: medición absoluta, alta resolución, alta precisión agregando conectividad Bluetooth, evitando los problemas con cables y conectores, mejorando y ampliando la conectividad.

La interfaz le proporcionará comodidad, permitiendo realizar mediciones en lugares difíciles de acceder.

Para utilizar la interfaz sólo debe enlazarla con algún dispositivo Bluetooth. Luego, solo resta pulsar el botón azul, para enviar el valor del Mitutoyo al dispositivo vinculado.

Calidad y Seguridad

Construida en inyección de plastico y al disponer de una bateria recargable la interfaz Bluetutoyo resulta indispensable para aquellos que necesiten una herramienta livina y portable.

La incorporacion de alta tecnologia y el ensanblado con componentes de alta calidad permite el envio de datos de forma confiable y segura.



Software

Gracias al Bluetutoyo los usuarios podrán ver las lecturas de medición directamente en su Smarthphone, tablet o PC logrando que las mediciones resulten más fáciles y ágiles.

La incorporación de alta tecnología y el ensamblado con componentes de alta calidad permite el envío de datos de forma confiable y segura.



Bluetutoyo cuenta con un puerto USB desde el que se puede recargar la bateria interna. Solo basta con conectar la interfaz a una computadora o a un cargador usb.

      • Tanto el software como la interfaz son multiplaforma, funcionan en Windows, Linux y OSx

      • El software permite exportar los datos en formato CSV para su posterior analisis con programas como Microsoft Excel u LibreOffice Calc y tambien permite generar un reporte PDF listo para imprimir.

Podemos modificar el software para que se adapte a sus necesidades.

    Utilice su tiempo de forma eficiente. Gracias al uso de la interfaz reducirá hasta en un 60% el tiempo requerido para realizar sus mediciones.

El diseño robusto de la interfaz no requiere ningun tipo de mantenimiento.
